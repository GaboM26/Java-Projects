POKER GAME in Java - Juego de Poker en Java
Made by/ Hecho por: David Gabriel Millares Bellido

English:
This poker game implements a simple game that a user can play against the computer by interacting with the PokerTest File.
By running command line arguments, one can test the different kinds of hands to not depend on randomness completely. 
When no command line arguments are given, then the game initiates normally. 

Español:
Este juego de poker implementa un juego simple que un usuario puede jugar contra la computadora por medio de el archivo PokerTest.
Si se pasan argumentos al programa podrá observar el funcionamiento de una carta en especifico. Si no se
pasan argumentos, entonces una partida normal se iniciara.
