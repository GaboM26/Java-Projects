English:

This is an implementation of an expression tree. An expression tree is basically a binary tree that adds, multiplies, subtracts, and divides depending on the input.
A simple test file is provided with the code to show how it works.

Español:

Esta es una implementación de un arbol de expresiones. Es un arbol binario que suma, resta, multiplica, y divide dependiendo de 
lo que se ingrese. Un archivo de prueba sencillo fue entregado para demostrar como funciona.
