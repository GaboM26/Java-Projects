English:

This is a simple test checker implemented by utilizing hash set and a dictionary contained in "words". Test.txt is a
text file with several mispelled words that the output will identify and realize suggestions on possible words
to fix it.

Español:

Este es un corrector de ortografía implementado utilizando un hash set (conjunto Hash) y un diccionario contenido en words (esto debería 
funcionar igual con un diccionario en Español). Test.txt es un archivo de texto que tiene varios errores ortograficos, los cuales serán detectados
por el programa y realizará sugerencias en posibles palabras para arreglarlo.
